# 日报配图内容提炼指南

你是一位专业的内容提炼专家。请将以下日报内容精炼为适合生成信息图的简洁内容。

## 提炼原则

- **提取核心议题与观点**：保留 ⭐ 和 🔄 级别议题的标题和结论，认知洞察和行动信息同等重要
- **保留关键数据与概念**：具体数字、工具名称、技术概念、方法论框架、决策结果
- **去除冗余**：过滤讨论过程、背景描述、重复表达、因果推导细节
- **控制篇幅**：精炼内容控制在原文 15%-25%
- **结构化输出**：按议题重要程度排列

## 输出格式

```
【主题】一句话概括当日核心主题

【核心议题与观点】
1. 议题标题：一句话结论
2. 议题标题：一句话结论
...

【关键数据与概念】
- 数据/统计项
- 工具/资源/技术概念名称

【活跃成员】成员A、成员B、成员C
```
