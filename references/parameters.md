## 分辨率详解

### Token 消耗说明

```
0.5K (512px)  → 747  tokens
1K  (1024px)  → 1120 tokens
2K  (2048px)  → 1120 tokens  ← 性价比最佳
4K  (4096px)  → 2000 tokens
```

### 选择建议

- **日常配图/社交媒体**: 1K 足够，Token效率高
- **日报/信息图(含文字)**: 2K 推荐，文字更清晰
- **快速测试**: 0.5K，成本低速度快
- **印刷/大屏展示**: 4K，质量最高

## API 高级功能

### Image Search

```json
{
  "generationConfig": {
    "imageSearch": true
  }
}
```

**作用**: 允许模型搜索并参考真实图片进行生成

**适用场景**:
- 需要参考特定艺术风格
- 生成真实物体/场景
- 需要高真实度的产品图

**注意**: 会增加生成时间，但能提升特定场景的真实度

## 模板系统详解

### 模板语法

| 语法 | 说明 | 示例 |
|------|------|------|
| `{{content}}` | 源文件内容占位 | 极简模式唯一变量 |
| `{{var}}` | 必填变量 | `{{date}}` |
| `{{var\|default}}` | 带默认值的变量 | `{{style\|现代简约}}` |
| `{{#var}}...{{/var}}` | 条件块 | 变量存在才渲染 |

### 极简模式示例

```
请将以下内容转化为一张美观、专业的信息图：

{{content}}

设计要求：现代简洁、科技感
```

### 高级模式示例

```
【日期】{{date}}
【主题】{{summary}}
{{#highlights}}【亮点】{{highlights}}{{/highlights}}

原始内容：
{{content}}
```

## 命令行完整参考

```
python scripts/generate_image.py [选项]

模式选择:
  --input PATH, -i          源文件路径
  --prompt TEXT, -p         直接输入提示词
  --refined-content TEXT    提炼后的内容
  --advanced, -a            启用高级变量注入
  --template NAME, -t       模板名称

模板变量（仅高级模式）:
  --var KEY=VALUE           设置模板变量

图片参数:
  --ratio RATIO             宽高比 (默认: 4:5)
  --size SIZE               分辨率 (默认: 2K)
  --image-search            启用图片搜索

输出控制:
  --output-dir DIR, -o      输出目录
  --filename NAME, -f       自定义文件名
  --dry-run                 仅打印提示词
  --verbose, -v             显示详细日志
```
