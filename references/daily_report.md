## 图片价值判断与分析深度

**第一步：判断图片价值**（根据上下文）

| 类型  | 判断标准                   | 处理方式        |
| --- | ---------------------- | ----------- |
| 高价值 | 技术截图、架构图、流程图、错误诊断、数据展示 | 必须理解完整逻辑链   |
| 中价值 | 新闻截图、配置界面、工具推荐         | 提取关键信息，简明描述 |
| 低价值 | 表情、无关截图、重复内容           | 一句话带过或不列入资源 |

**第二步：高价值图片分析要求**

- 必须理解：问题→分析→解决→固化的完整链条

- 必须关联：谁发的、为什么发、回应什么问题

**示例对比**：

- ❌ 低质量："截图显示了evolver报错"

- ✅ 高质量："截图展示evolver自我进化闭环：识别问题(Exit code 1)→分析根因(CMD语法不兼容bash)→实施修正(Python跨平台方案)→固化经验(写入MEMORY.md成为GEP evolution event)"
