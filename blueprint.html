09 / Privacy & Security
gitignore Protection
Excludes: config.local.md, .env, memory/, temp/, reports/, __pycache__/
No sensitive data enters version control.
privacy_scanner.py
Pre-commit hook. Scans staged files for:
API keys · Group IDs · Phone numbers · Local file paths
Blocks commit if violations detected.
config_loader.py — Privacy Guard
Raises PrivacyError if unsubstituted placeholders like ${...} are detected in loaded config. Prevents template leakage into runtime.
